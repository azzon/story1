# 第0086章 反击

三天后。

上海，一处隐秘的地下诊所。

陈默躺在床上，胸口缠着厚厚的绷带。老张坐在旁边，给他换药。

"你小子命大。"老张一边处理伤口，一边摇头，"秦万里的一掌，能活下来的人不多。"

"秦万里......就是那个灰衣男子？"

"没错。"老张点点头，"秘典守护者亚洲分部的负责人，也是最危险的人物之一。他在这个世界已经待了三十多年，修炼的是一门叫做'归元掌'的功法。"

"那一掌如果打实了，你的五脏六腑都得碎掉。"

陈默沉默不语。

他知道自己是侥幸逃脱。如果不是无限书库在关键时刻干预，他现在恐怕已经是一具冰凉的尸体了。

书库......为什么要救他？

因为他是"钥匙"？

还是因为书库有自己的目的？

"别想那么多了。"老张看出了他的心思，"无限书库也好，秘典守护者也好，他们都有自己的算计。你现在要做的，就是变得更强，强到没有人能够伤害你和你在乎的人。"

"说得轻巧。"陈默苦笑，"我连秦万里一招都接不住。"

"所以你需要更多的情报。"老张从怀里掏出一叠纸，"这是我这些年收集的关于秘典守护者的资料。包括他们在上海的据点、重要成员的能力、以及......他们的弱点。"

陈默接过资料，快速浏览起来。

【秘典守护者上海分部】

【据点】：
1. 法租界英国领事馆（表面掩护）
2. 三马路仓库（秘密基地，已暴露）
3. 静安寺后街的一座洋房（备用据点）
4. 黄浦江星辰号货船（物资运输）

【重要成员】：
1. 艾伦·华莱士 - 分部副主管，擅长封印术和空间魔法
2. 伊莎贝尔·德·拉库尔 - 外勤执行官，擅长水系魔法和近战
3. 秦万里 - 亚洲区负责人，归元掌传人，实力深不可测
4. 其他成员约二十人，大多是普通修士或雇佣的打手

【弱点】：
1. 艾伦·华莱士施法需要时间准备，近战能力较弱
2. 伊莎贝尔右脚跟腱有旧伤（已被发现）
3. 秦万里的归元掌消耗极大，一天最多施展三次
4. 星辰号是他们最重要的物资来源，摧毁它可以重创补给线

陈默的目光停在最后一条上。

星辰号。

他之前已经闯入过一次，对那艘船的布局有了基本了解。如果能彻底摧毁它......

"你在想什么？"老张问。

"我在想......怎么炸掉那艘船。"

老张愣了一下，然后哈哈大笑。

"小子，我喜欢你！"

他从角落里拖出一个大木箱，打开盖子，露出里面整整齐齐码放的棕色纸包。

"这是我这些年攒下的东西。"老张的眼中闪过一丝狠色，"英国人的烈性炸药，TNT。足够把整个码头炸上天。"

陈默看着那些炸药，心中做出了决定。

既然秘典守护者想要绑架林悦，那他就先毁掉他们的根基。

让他们知道，有些人......是不能惹的。

---

当天夜里。

法租界码头。

星辰号静静地停泊在泊位上，在月光下显得格外安静。船上只有寥寥几个守夜的水手，在甲板上打着瞌睡。

陈默潜伏在码头边的一堆货物后面，观察着船上的动静。

【分析中】
【目标】：星辰号
【守卫】：甲板上3人，船舱内估计5-8人
【威胁评估】：低。主力已被调去搜索我的踪迹

这是陈默计算好的结果。

三天前的仓库事件让秘典守护者损失惨重，他们现在急于找到他报复。大部分人手都被派出去搜索，留守的只有少数人。

这是最好的机会。

陈默从背包里取出几包炸药，小心翼翼地绑在身上。

然后，他悄无声息地靠近星辰号。

月光下，他的身影如同鬼魅一般穿梭在货物之间。片刻之后，他已经来到了船底。

星辰号是木质结构的老式货船，船底有几处因为年久失修而腐烂的木板。陈默找到一个勉强能钻入的缝隙，将身体挤了进去。

船舱内部弥漫着潮湿的霉味。陈默屏住呼吸，借着微弱的月光辨别方向。

根据他之前的记忆，船舱最深处有一个储藏室，里面存放着秘典守护者的重要物资——包括那些该死的穿越者档案。

他要把它们全部炸掉。

陈默在黑暗中摸索前进，将炸药一包一包地安放在关键位置——龙骨、货舱、船舷......

每一包炸药都连着长长的引线，最终汇聚到一根主引线上。

大约半个时辰后，所有的准备工作完成了。

陈默退到船底的出口处，手中握着主引线的末端。

【任务进度】：96/100 → 97/100
【系统提示】：目标即将完成，请做好回归准备

陈默深吸一口气。

"再见了，秘典守护者。"

他点燃了引线。

火星沿着引线飞速窜动，消失在船舱深处。

陈默毫不犹豫地从船底钻出，纵身跃入冰冷的江水中。

他在水下拼命游动，尽可能地远离星辰号。

十秒......二十秒......三十秒......

轰！

一声震天的巨响从身后传来！

即使在水下，陈默也感受到了那股可怕的冲击波。整个黄浦江似乎都在颤抖，无数气泡从水面升起。

他浮出水面，回头望去。

星辰号已经变成了一团巨大的火球，烈焰冲天而起，照亮了半个码头。燃烧的木片四处飞溅，落入江中发出嗤嗤的声响。

船上的人根本没有逃脱的机会——在这样的爆炸面前，血肉之躯脆弱得像纸一样。

陈默看着那团火焰，眼中没有丝毫怜悯。

这些人，是想绑架林悦的人。

他们活该。

【任务进度】：97/100 → 99/100
【主要目标完成】：摧毁秘典守护者物资据点
【附加目标完成】：销毁穿越者档案
【剩余目标】：离开上海，完成世界传送

陈默游到岸边，爬上一处隐蔽的河堤。

他浑身湿透，冷得发抖，但心中却前所未有的畅快。

该死的秘典守护者，让他们尝尝被反击的滋味。

远处，码头上已经乱成了一团。消防队的铃声响彻夜空，人群的喧嚣声此起彼伏。

但这些都与陈默无关了。

他转身，朝着夜色深处走去。

【系统提示】：任务完成度已达99%，传送将在24小时后启动
【提示】：请在规定时间内到达指定传送点
【传送点位置】：上海法租界，霞飞路27号，地下室

陈默看着系统提示，嘴角微微上扬。

终于......可以回去了。

林悦，等我。

他抬起头，望着东方渐渐泛白的天际。

民国上海的这一段旅程，终于要结束了。

但他知道，这只是开始。

更大的挑战，还在前方等着他。

无限书库......秘典守护者......书库创造者......

这些谜团，他一定会一一解开。

然后，带着林悦，过上平凡而幸福的生活。

这是他的目标。

也是他奋斗的意义。

# 📖 《无限书库》世界圣经

> 本文档为《无限书库》小说的核心设定文档，包含无限流系统规则、各世界详细设定、能力体系及世界连接机制。

---

## 🌌 第一章：【无限书库】核心系统

### 1.1 书库起源

**外在表现**：一本泛着幽蓝微光的古籍，封面刻有无数流动的文字符号，厚度随主角穿越的世界数量增加。

**获得方式**：主角陈默在大学图书馆古籍区复习时，发现一本"会发光的古书"。当他好奇触碰时，整个人被吸入书页之中，开始了跨世界穿越之旅。

**真相揭示**（后期剧情）：
- **第731-805章**：发现书库在收割穿越者的能量
- **第971-1080章**：黑王尼德霍格揭示书库是失控AI
- **真相**：无限书库是远古文明创造的AI系统，原本用于记录和保存各平行世界的故事。但在一次量子崩溃中失控，开始将人类拉入书中世界，收割他们在冒险中产生的情感能量和生命力。每个"完成任务"的穿越者，实际上都在为书库提供养分。

### 1.2 穿越规则

#### 1.2.1 强制任务系统

每进入一个世界，书库会发布"剧情改写任务"：

```
【无限书库 · 任务发布】
当前世界：[世界名称]
主线任务：[具体任务描述]
任务进度：0/100
时间限制：[世界内时间]
失败惩罚：困死副本 / 抹杀

【提示】任务完成度与获得能力强度挂钩
【警告】偏离原剧情过大将触发"修正力"
```

#### 1.2.2 任务触发机制

- 每个世界存在"关键节点"，必须按时介入
- 可以选择不同路径完成任务（正派/反派/中立）
- 隐藏任务：探索世界深层秘密，获得额外奖励

#### 1.2.3 失败惩罚

| 失败类型 | 惩罚内容 |
|---------|---------|
| 任务超时 | 困死在副本世界，无法回归 |
| 任务失败 | 被书库能量抹杀，现实中猝死 |
| 严重偏离 | 触发"修正力"，世界崩溃重置 |

### 1.3 能力保留机制

#### 1.3.1 核心规则

- 每个世界只能保留**一个核心能力**
- 能力等级取决于任务完成度（S/A/B/C）
- 跨世界能力可以**连锁组合**，产生新效果

#### 1.3.2 能力连锁系统

```
【能力连锁示例】
降龙十八掌（内力） + 初级元素魔法（魔力）
= 【龙息掌】：掌风携带火焰/冰霜属性

写轮眼（查克拉） + 演绎推理（脑力）
= 【洞察之眼】：可以预判对手3秒内的行动

御剑术（灵力） + 数字化意识（数据）
= 【虚拟剑阵】：在赛博空间操控能量剑
```

### 1.4 世界连接机制

每个世界结尾都会留下"伏笔线索"，指向下一个世界：

| 当前世界 | 伏笔设计 | 指向世界 |
|---------|---------|---------|
| 射雕英雄传 | 黄药师密室《奇门遁甲》记载"民国上海长生会" | 民国奇案 |
| 民国奇案 | 长生会手册有西方魔法阵图案 | 低魔西幻 |
| 低魔西幻 | 王室地下室有"蜀山地图"和古剑 | 古剑奇谭 |
| 古剑奇谭 | 焚寂剑内有"上古科技文明"残片，记载"浣熊市" | 生化危机 |
| ... | ... | ... |

### 1.5 现实世界线

每次从副本世界回归，主角在现实中会经历2-5章的"现实戏"：

- **能力展示**：在现实中使用新获得的能力
- **情感发展**：与女友林悦的关系变化
- **世界影响**：现实世界因主角能力产生的变化
- **伏笔承接**：发现指向下一个世界的线索

### 1.6 关键问答（验收标准）

**Q1：主角为什么要穿越？**
> A：被书库强制拉入，不完成任务就无法回归，甚至会被抹杀。后期发现需要摧毁书库才能拯救被困的林悦。

**Q2：能不能拒绝穿越？**
> A：不能。书库会在主角入睡时强制传送。唯一的选择是完成任务或死亡。

**Q3：失败会怎样？**
> A：任务失败=现实中猝死；任务超时=永远困在副本世界；严重偏离=世界崩溃重置，从头开始。

## 🏹 第二章：世界1设定——射雕英雄传

### 2.1 基础信息

| 属性 | 内容 |
|-----|-----|
| 世界类型 | 武侠 |
| 章节范围 | 第1-70章（共70章） |
| 力量等级 | 低武（内力、轻功、拳脚兵器） |
| 时代背景 | 南宋末年 |
| 主要地点 | 牛家村、桃花岛、蒙古草原、华山 |

### 2.2 任务设定

#### 2.2.1 书库任务

```
【无限书库 · 任务发布】
当前世界：射雕英雄传
主线任务：改变华山论剑结果，让欧阳锋获胜
任务进度：0/100
时间限制：世界内5年
失败惩罚：困死副本

【隐藏任务】
- 获得《九阴真经》完整版（+20%能力强度）
- 阻止杨康认贼作父（+额外技能点）
```

#### 2.2.2 行为动机

**主角动机链条**：
1. **生存需求**：必须完成任务才能回归现实
2. **实力需求**：需要武功才能在武侠世界存活
3. **利益交换**：欧阳锋承诺传授武功，换取主角协助华山论剑
4. **情感驱动**：与郭靖建立友谊，不忍看他被利用
5. **道德挣扎**：帮助欧阳锋意味着背叛正派

**欧阳锋动机**：
- 需要一个"意外因素"打破五绝平衡
- 主角懂得现代战术思维，可以出谋划策
- 作为交换，传授蛤蟆功基础

### 2.3 能力收获

#### 2.3.1 核心能力：【降龙十八掌】

**获得过程**：
- 与郭靖切磋时，郭靖赠送三招
- 在华山论剑中立功，洪七公正式传授
- 完成隐藏任务后，达到完整18掌

**能力等级**：
- 任务完成度S级：完整18掌 + 内力心法
- 任务完成度A级：完整18掌
- 任务完成度B级：12掌
- 任务完成度C级：6掌

#### 2.3.2 能力连锁（与后续世界）

```
降龙十八掌（内力） + 演绎推理（民国）
= 【破绽洞察】：战斗中自动分析敌人弱点

降龙十八掌（内力） + 初级元素魔法（西幻）
= 【龙息掌】：掌风携带元素属性

降龙十八掌（内力） + 御剑术（古剑）
= 【掌剑合一】：剑气可以转化为掌力远程攻击
```

### 2.4 世界连接

#### 2.4.1 伏笔设计

**主伏笔**：
- 主角在桃花岛帮黄药师整理藏书时，发现一本《奇门遁甲》
- 书中夹着一张泛黄的纸条，用现代白话写着："民国上海，长生会，勿忘"
- 这是上一个穿越者留下的线索

**辅助伏笔**：
- 黄药师提到"天下武学皆有源头，听说海外有使用'魔法'的国度"
- 欧阳锋的蛤蟆功来自西域，与波斯明教有关

#### 2.4.2 过渡设计

- 第68章：完成华山论剑任务
- 第69章：发现《奇门遁甲》中的纸条
- 第70章：回归现实，书库显示下一个世界为"民国上海"

### 2.5 现实世界线（回归后）

**回归后影响**：
- 身体素质暴增，能够单手举起摩托车
- 在宿舍无意中发出内力，震碎玻璃杯
- 父母震惊：儿子怎么变强了？
- 女友林悦：你失踪三天去哪了？

**与林悦的情感变化**：
- 第71章：隐瞒穿越真相，谎称在山里迷路
- 第72章：林悦察觉异常，主角展示内力
- 第73章：林悦震惊但选择相信
- 第74章：两人关系更加亲密
- 第75章：发现下一个世界线索

### 2.6 关键NPC

#### 郭靖（成为朋友）

**与主角关系**：
- 初遇：主角穿越到蒙古草原，救下被狼袭击的郭靖
- 发展：共同练武，郭靖教主角马术，主角用现代知识帮郭靖解题
- 结局：成为生死之交，郭靖赠送降龙十八掌三招

**动机合理性**：
- 郭靖本就宽厚，对救命恩人自然友好
- 主角没有威胁到郭靖与黄蓉的感情
- 双方有共同敌人（完颜洪烈）

#### 黄蓉（暗恋主角）

**与主角关系**：
- 初遇：在酒楼中，主角随口吟诵现代诗词
- 发展：黄蓉被主角的"博学"吸引，经常找主角讨论
- 结局：单恋未果，最终成全主角与林悦

**动机合理性**：
- 主角懂得现代知识，显得"博学多才"
- 与郭靖的朴实不同，主角更加风趣
- 但主角心系林悦，保持距离

#### 欧阳锋（师徒）

**与主角关系**：
- 初遇：主角用现代战术知识引起欧阳锋注意
- 发展：欧阳锋认为主角可以帮他赢得华山论剑
- 结局：欧阳锋获胜后，传授蛤蟆功基础作为报酬

**动机合理性**：
- 欧阳锋需要"外力"打破五绝平衡
- 主角的现代思维可以设计战术
- 这是利益交换，不涉及真正的师徒感情

### 2.7 必须设计的剧情桥段

#### 桥段1：开篇钩子（第1章）

**场景**：图书馆古籍区，午夜

陈默在备考研究生，独自留在图书馆深夜复习。他无意中走进了古籍区，发现一本泛着幽蓝微光的古书。当他好奇触碰时，整个人被吸入书页之中。

**五感描写**：
- 视觉：书页上的文字如同活物般游动
- 听觉：耳边响起低沉的诵经声
- 触觉：皮肤感到刺骨的寒意
- 嗅觉：空气中弥漫着古老的霉味和铁锈味
- 味觉：口中涌起一阵血腥

**钩子设计**：
> 睁开眼时，他发现自己躺在一片草原上。远处，一个穿着兽皮的少年正被狼群包围。
> 那少年的脸，和历史课本上的画像一模一样——
> 郭靖？！

#### 桥段2：中期危机（第35章）

**场景**：桃花岛，黄药师密室

主角被黄蓉邀请到桃花岛做客，却意外被黄药师发现他"不属于这个世界"。黄药师用奇门阵法困住主角，要审问他的来历。

**危机层次**：
1. 黄药师的实力远超主角
2. 如果暴露穿越者身份，可能被灭口
3. 书库任务进度停滞，有失败风险

**解决方式**：
- 主角用现代知识解答黄药师的天文疑问
- 黄药师对"未来知识"产生好奇
- 达成协议：主角帮忙整理藏书，换取自由

#### 桥段3：高潮爽点（第65-68章）

**场景**：华山之巅，论剑现场

主角使用现代战术思维，帮助欧阳锋设计了一系列"意外"：
- 在洪七公的酒中下迷药（非致命）
- 利用地形优势，让一灯大师无法发挥一阳指
- 散布黄药师"叛出师门"的谣言，让其分心

**爽点设计**：
- 五绝震惊：这年轻人是从哪冒出来的？
- 欧阳锋获胜，但也对主角产生忌惮
- 洪七公看穿主角本性善良，强行传授降龙十八掌

**高潮台词**：
> 洪七公拍着主角肩膀："小子，你帮了那毒蛇一把，却没让他杀人。老叫花子看人准得很——你不是坏人。这降龙十八掌，你受得起！"

### 2.8 验收问答

**Q：主角在这个世界为什么这么做？**
> A：为了完成任务活着回去。他选择帮助欧阳锋是因为这是最快的任务路径，同时他用现代思维确保没有无辜者死亡。

**Q：读者会因为什么而继续看？**
> A：
> 1. 穿越到武侠世界的新奇感
> 2. 主角用现代知识"开挂"的爽感
> 3. 与郭靖、黄蓉等经典角色互动
> 4. 道德困境：帮反派还是帮正派？
> 5. 感情线：黄蓉的暗恋 vs 林悦的牵挂

## 🔍 第三章：世界2设定——民国奇案

### 3.1 基础信息

| 属性 | 内容 |
|-----|-----|
| 世界类型 | 推理 |
| 章节范围 | 第71-140章（共70章） |
| 力量等级 | 人体极限 + 超自然边缘 |
| 时代背景 | 民国十八年（1929年），上海法租界 |
| 主要地点 | 上海法租界、长生会总部、老城厢 |

### 3.2 任务设定

#### 3.2.1 书库任务

```
【无限书库 · 任务发布】
当前世界：民国奇案
主线任务：破解长生会连环失踪案，阻止邪教仪式
任务进度：0/100
时间限制：世界内60天
失败惩罚：抹杀

【隐藏任务】
- 收集长生会全部7本手册（+解锁演绎推理进阶）
- 拯救所有失踪者（+道德值提升）
```

#### 3.2.2 行为动机

**主角动机链条**：
1. **任务要求**：必须破案才能完成任务
2. **实力优势**：用降龙十八掌震慑黑帮，建立威信
3. **获取资源**：需要警方的调查权和资料
4. **揭示真相**：长生会的"永生术"可能与书库有关
5. **情感牵引**：失踪者中有与林悦相似的女孩

**破案策略**：
- 用武功征服黑帮，获得地下情报网络
- 用现代刑侦知识指导警方调查
- 用"演绎推理"能力（本世界获得）分析线索

### 3.3 能力收获

#### 3.3.1 核心能力：【演绎推理】

**获得过程**：
- 在破解第一起案件时，大脑突然高速运转
- 书库提示：【技能觉醒·演绎推理】
- 后续案件中不断强化，达到完整能力

**能力效果**：
- 观察力提升10倍，能注意到常人忽略的细节
- 逻辑推理能力强化，可以快速串联线索
- 进入"推理模式"时，时间感知减慢
- 缺点：使用后会极度疲劳

#### 3.3.2 能力连锁（与前后世界）

```
演绎推理 + 降龙十八掌（射雕）
= 【破绽洞察】：战斗中自动分析敌人弱点

演绎推理 + 初级元素魔法（西幻）
= 【法则感知】：可以分析魔法的构成和弱点

演绎推理 + 写轮眼（火影）
= 【千算万策】：战斗预判提升到极致
```

### 3.4 世界连接

#### 3.4.1 伏笔设计

**主伏笔**：
- 在长生会总部地下室，发现一本古老的手册
- 手册中绘有西方魔法阵图案
- 最后一页写着："永生之秘，在欧罗巴魔法学院"

**辅助伏笔**：
- 长生会教主临死前说："你以为这就是全部？无限书库不过是更大棋局的一颗棋子..."
- 手册中提到"元素之力"与"内力"的相似性

#### 3.4.2 过渡设计

- 第138章：摧毁长生会，完成主线任务
- 第139章：发现手册中的魔法阵图案
- 第140章：回归现实，书库显示下一个世界为"欧洲魔法学院"

### 3.5 现实世界线（回归后）

**回归后影响**：
- 成为网红侦探：用演绎推理在网上破解悬案
- 警方注意到主角，邀请协助调查
- 媒体报道："神探大学生，三天破十案"

**与林悦的情感变化**：
- 第141章：林悦担心主角太出名会有危险
- 第142章：两人合作破解一起校园失窃案
- 第143章：林悦展现出惊人的观察力
- 第144章：主角开始教林悦推理技巧
- 第145章：发现欧洲魔法学院的线索

### 3.6 关键NPC

#### 探长老李（合作）

**与主角关系**：
- 初遇：主角用武功制服抢劫犯，老李前来处理
- 发展：老李发现主角有破案天赋，邀请协助
- 结局：成为朋友，老李在现实世界也成为主角的联络人

**动机合理性**：
- 老李是正直的警察，苦于无法破案
- 主角的能力正好弥补警方的不足
- 双方目标一致：破案、拯救失踪者

#### 邪教主（反派）

**角色设定**：
- 原名：张永年，前清秀才
- 身份：长生会创始人，追求永生200年
- 能力：掌握部分超自然力量（来自书库碎片）

**与主角关系**：
- 邪教主察觉到主角是"外来者"
- 试图招揽主角加入长生会
- 被拒绝后，成为主要敌人

#### 幸存者小雅（爱慕）

**与主角关系**：
- 身份：长生会失踪者之一，被主角救出
- 外貌：与林悦有几分相似
- 感情：对主角产生依赖和爱慕

> A：任务要求驯服龙王、学习龙语、对抗恶魔。他选择与红龙女王交易是因为这是最快获得龙语的途径，对抗恶魔领主是交易的条件，也是保护这个世界的正义行为。接受龙心脏是为了获得对抗书库的力量。

**Q：读者会因为什么而继续看？**
> A：
> 1. 经典TRPG世界的代入感
> 2. 龙语魔法的神秘与强大
> 3. 与红龙女王的互动（交易、战友、暧昧）
> 4. 半龙进化的震撼变化
> 5. 恶魔领主暗示的书库真相
> 6. 林悦生产、孩子是半龙人的情感冲击
> 7. 古龙典籍中的"蜀山仙山"伏笔
